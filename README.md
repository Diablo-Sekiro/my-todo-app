# Task Tracker Uygulaması

Günlük üretkenliği artırmak için geliştirilmiş, modern ve kullanıcı dostu bir görev yönetim uygulaması.

## Özellikler
- **Görev Yönetimi**: Görev ekleme, düzenleme (güncelleme), silme ve tamamlandı olarak işaretleme.
- **Veri Saklama**: `localStorage` kullanarak verileri tarayıcıda saklar; sayfayı yenileseniz bile görevleriniz kaybolmaz.
- **Filtreleme**: Görevleri "Tümü", "Aktif" ve "Tamamlananlar" olarak hızlıca filtreleme.
- **İstatistik Paneli**: Toplam, devam eden ve tamamlanan görev sayısını gösteren şık bir dashboard.

## Kullanılan Teknolojiler
- **Frontend**: ReactJS, Vite
- **Stil**: Pure CSS
- **Yayına Alma**: Netlify
